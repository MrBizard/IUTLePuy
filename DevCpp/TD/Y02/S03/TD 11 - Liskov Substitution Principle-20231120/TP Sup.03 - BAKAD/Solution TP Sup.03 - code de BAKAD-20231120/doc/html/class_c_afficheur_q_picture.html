Description détaillée

Classe d'affichage des formes en utilisant Qt.

Le principe de cette classe est de faire le rendu en mémoire dans un QPicture. Ce QPicture peut ensuite être affiché dans un QPaintDevice (par exemple un QWidget au moment de son paintEvent) en appelant la fonction drawIt.
