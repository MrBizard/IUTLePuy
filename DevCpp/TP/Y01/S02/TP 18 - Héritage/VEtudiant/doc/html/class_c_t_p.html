Description détaillée

Classe "métier" du TP.

Cette classe concentre toutes les actions du programme nécessaires pour le fonctionnement du programme. Les méthodes publiques sont les points d'entrées événementiels qui sont appelés par le programme principal en réponse des interactions de l'utilisateur.

La classe CTP est déclarée dans le fichier TP.h.

#include "TP.h"

◆ ModifierDernierArcStartAngle()

void ModifierDernierArcStartAngle ( const LibGraph2::CPoint &  pt)

Définit l'angle de départ de l'arc.

L'angle de départ de l'arc est calculé à partir du point pt passé en paramètre. L'angle calculé est l'angle que réalise le vecteur qui part du centre de l'ellipse vers le point pt avec l'horizontale. Pour calculer cet angle, il suffit d'appliquer l'équation suivante :

\[ a = \text{ atan2 } \left( \vec{ v }_y, \vec{ v }_x \right) * \frac{ 180 }{ \pi } \]

où \( \vec{v} \) est le vecteur du centre de l'ellipse vers le point pt.

Paramètres
ptPoint de destination du vecteur définissant l'angle de départ de l'arc.
A faire:
Modifier le dernier arc créé pour lui affecter son angle de départ en suivant les indications précédentes.

◆ ModifierDernierArcSweepAngle()

void ModifierDernierArcSweepAngle ( const LibGraph2::CPoint &  pt)

Définit l'angle de balayage de l'arc.

L'angle de balayage de l'arc est calculé à partir du point pt passé en paramètre. L'angle calculé est l'angle que réalise le vecteur qui part du centre de l'ellipse vers le point pt avec l'horizontale auquel on retranche l'angle de départ. Pour calculer cet angle, il suffit d'appliquer les équations suivantes :

\[ a_{temp} = \text{ atan2 } \left( \vec{ v }_y, \vec{ v }_x \right) * \frac{ 180 }{ \pi } - a_{start} \]

\[ a=\begin{cases} a_{temp}, & \text{ si }a_{temp}>=0 \text{ , } \\ a_{temp} + 360, & \text{ sinon} \end{cases} \]

où \( \vec{v} \) est le vecteur du centre de l'ellipse vers le point pt et \( a_{start} \) est l'angle de départ de l'arc.

Paramètres
ptPoint de destination du vecteur définissant l'angle de balayage de l'arc.
A faire:
Modifier le dernier arc créé pour lui affecter son angle de balayage en suivant les indications précédentes.
