Description détaillée

Cette classe regroupe toutes les fonctionnalités du niveau Avancé d'une fenêtre graphique LibGraph 2.

Remarques
Pour une plus grande portabilité du code entre les niveaux Avancé et Expert, vous pouvez utiliser la classe ILibGraph2, qui est un alias vers cette classe.
