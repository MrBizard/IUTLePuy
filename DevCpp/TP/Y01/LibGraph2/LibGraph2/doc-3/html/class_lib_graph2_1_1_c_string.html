Description détaillée

L'utilisation de cette classe est transparente car elle comporte les fonctions de convertion depuis et vers les chaines de caratères de type char* ou wchar*.
Son utilité principale est de rendre transparent les problèmes de conversion Unicode.

Remarques
Les fonctions nécessitant des CString en entrée peuvent donc accepter automatiquement des chaines de caractères de type char* ou wchar*.
