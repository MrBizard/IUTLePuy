◆ ARGB

typedef DWORD LibGraph2::ARGB

Ce type de données permet de stocker une couleur selon le mode RVB avec gestion de la transparence. Utilisez la fonction MakeARGB() pour créer des couleurs.

Remarques
Il s'agit d'un entier non signé sur 32 bits (unsigned long).
