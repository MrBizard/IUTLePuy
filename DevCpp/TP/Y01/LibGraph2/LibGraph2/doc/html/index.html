Introduction

Présentation

LibGraph 2 est une bibliothèque graphique simplifiée. Elle permet par un jeu de fonctionnalités d'afficher une fenêtre graphique et d'y afficher différentes formes.

Capture d'écran d'un logiciel entièrement développé à l'aide de LibGraph 2

Niveaux d'utilisation

La bibliothèque s'adapte au niveau d'expérience de l'utilisateur selon 5 niveaux (Cliquez sur l'une des descriptions pour accéder à la documentation du niveau) :

  • NIVEAU 0 : Novice : Aucun pointeur, aucun objet, pas de passage par référence, seulement des appels de fonctions
  • NIVEAU 1 : Débutant : Des passages par références, pas de pointeurs, pas d'objet
  • NIVEAU 2 : Intermédiaire : Des passages par références, des pointeurs, pas d'objet
  • NIVEAU 3 : Avancé : Quelques objets
  • NIVEAU 4 : Expert : (niveau par défaut) Tout objet

Installation

Quelque soit le niveau d'expérience choisi, l'installation de la bibliothèque suit toujours la même procédure :

Création du projet Visual Studio

Créez un nouveau projet C++ Visual Studio de type Projet vide pour la plateforme Windows :

Copie des fichiers

  • Copier les répertoires lib (qui se trouve dans le sous répertoire v142 ) et inc dans le répertoire de votre projet (à côté du fichier *.vcxproj).
  • Copier les fichiers lib\Release\LibGraph2.dll et lib\Debug\LibGraph2.dll respectivement dans les répertoires x64\Release et x64\Debug de votre solution :
    Note
    Les répertoires x64\Debug et x64\Release sont les répertoires où votre programme exécutable sera créé. Ces répertoires sont créés lors de la compilation. Comme vous n'avez pas encore compilé votre projet, ils n'existent probablement pas. Vous devez donc les créer vous-même.

Configuration du projet

  • Ouvrir le gestionnaire de configuration (menu Générer / Gestionnaire de configurations...) puis dans la liste Plateforme de la solution active, choisissez <Modifier...> puis supprimez la plateforme x86 :
    Gestionnaire de configuration
  • Ouvrir les propriétés du projet (Dans Visual Studio, faites un clic droit sur le projet puis Propriétés)
  • Choisissez "Toutes les configurations" dans la liste déroulante "Configuration :" puis dans la page "C/C++" / "Répertoire VC++", ajoutez ";$(ProjectDir)inc" dans le champ "Répertoires Include" et ";$(ProjectDir)lib\$(Configuration)" dans le champs "Répertoires de bibliothèques" :
  • Dans la page "Editeur de liens" / "entrée", ajoutez "LibGraph2.lib" au champ "Dépendances supplémentaires" :
  • Dans la page "Editeur de liens" / "Système", définissez le Sous-système comme "Windows (/SUBSYSTEM:WINDOWS)" :
  • Validez. --> LibGraph2 est maintenant configuré pour votre projet. Choisissez maintenant un niveau d'utilisation pour un exemple de programme principal.
