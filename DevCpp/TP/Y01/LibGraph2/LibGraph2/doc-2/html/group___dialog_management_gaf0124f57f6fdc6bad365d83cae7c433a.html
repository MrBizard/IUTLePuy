LIBGRAPH2_API bool LibGraph2::guiGetPenStyle ( ARGB &  color,
float &  fWidth,
pen_DashStyles &  style 
)

Affiche la boîte de dialogue de sélection des attributs de traits, permettant de choisir la couleur, l'épaisseur et le style d'un trait.

Boîte de dialogue de sélection des attributs de traits
Paramètres
[in,out]colorEn entrée, couleur à définir dans la boîte de dialogue. En sortie, couleur sélectionnée dans la boîte de dialogue si l'utilisateur clique sur Ok.
[in,out]fWidthEn entrée, épaisseur de trait à définir dans la boîte de dialogue. En sortie,
épaisseur choisie si l'utilisateur clique sur Ok.
[in,out]styleEn entrée, style de trait à définir dans la boîte de dialogue. En sortie,
style choisi si l'utilisateur clique sur Ok.
Renvoie
true si l'utilisateur a cliqué sur le bouton Ok, false sinon.
